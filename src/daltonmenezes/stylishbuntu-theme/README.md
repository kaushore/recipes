# Stylishbuntu Theme

This recipe will install and configure:

- El General icons
- Flatabulous theme

# Usage

| Command | Support |
| --- | --- |
| `uni bake daltonmenezes/stylishbuntu-theme` | Ubuntu 16.04 (Unity) |

# Uninstallation

| Command |
| --- |
| `uni bake daltonmenezes/stylishbuntu-theme-remove` |
